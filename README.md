# FetchApp

* 根据应用的id查询应用的gem
* 应用限于安卓应用和苹果应用


``
  FetchApp.fetch_app('ios','**.**')
  FetchApp.fetch_ios('**.**')
  FetchApp.fetch_android('**')
``

## Installation

Add this line to your application's Gemfile:

    gem 'fetch_app'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install fetch_app

## Usage

One Gem To Fetch Android or IOS
